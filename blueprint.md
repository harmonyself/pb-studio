# Blueprint: 유튜브 썸네일 생성기

## 1. 프로젝트 개요

'꿈의 그라운드 경남' 스타일의 유튜브 강연 썸네일을 쉽고 빠르게 제작할 수 있는 웹 애플리케이션입니다. 사용자는 강연자 사진, 배경, 로고, 텍스트 등 각 요소를 직접 업로드하고 편집하여 완성도 높은 썸네일을 생성하고 다운로드할 수 있습니다.
**주요 업데이트:** 구글 애드센스 승인을 위한 '고부가가치 콘텐츠(Publisher Content)' 섹션이 추가되어, 단순 도구를 넘어선 정보성 웹사이트로 확장되었습니다.

## 2. 핵심 기능

- **템플릿 기반 제작:** 제공된 '꿈의 그라운드 경남' 썸네일 템플릿을 기반으로 합니다.
- **실시간 캔버스 편집:** 모든 변경사항은 캔버스에 실시간으로 반영되어 직관적인 편집이 가능합니다.
- **요소별 커스터마이징:**
    - **강연자 이미지:** 좌측 배치, 위치/크기 조절 가능. **AI 자동 배경 제거 기능 탑재.**
    - **배경 이미지:** 3장 합성(블렌딩) 지원. **랜덤 배경 추천 기능**으로 손쉽게 배경 설정 가능.
    - **로고 이미지:** 우측 상단 배치, 위치/크기 조절 가능.
    - **강조 문구:** 괄호 `( )`를 사용한 특정 단어 색상 강조 기능.
- **정보성 콘텐츠 제공:** 썸네일 제작 가이드, 디자인 원칙, 마케팅 전략 등을 담은 전문적인 아티클 섹션 제공.

## 3. 기술 스택

- **HTML5:** Semantic Markup (`<article>`, `<section>`, `<header>` 등 적극 활용)
- **CSS3:** Flexbox, Grid, CSS Variables, Dark Mode 지원, 반응형 디자인
- **JavaScript (ES6+):** Canvas API, Promise 기반 비동기 처리, `@imgly/background-removal` 라이브러리 연동

## 4. 디자인 및 스타일 가이드

- **레이아웃:**
    - 상단: 도구 (좌측 컨트롤 패널 + 우측 캔버스 미리보기)
    - 하단: 정보성 아티클 (블로그 스타일의 가독성 높은 타이포그래피)
- **색상 팔레트:**
    - Primary: `#FF0000` (유튜브 레드)
    - Background: `#121212` (다크 모드 기본) / `#ffffff` (라이트 모드)
    - Highlight: `#FFFF00`
- **폰트:** `Pretendard` (가독성 및 심미성 확보)

## 5. 구현 현황 (완료)

1.  **HTML 구조:**
    - 도구 섹션과 정보 섹션의 명확한 구분.
    - SEO 최적화를 위한 메타 태그 및 시맨틱 태그 적용.
    - 불필요한 "준비중" 요소 제거 및 기능화.

2.  **기능 구현:**
    - **배경 제거:** 이미지 업로드 시 자동 실행.
    - **랜덤 배경:** 버튼 클릭 시 샘플 이미지 중 3장을 랜덤으로 선택하여 자동 합성.
    - **캔버스 드로잉:** 텍스트 외곽선, 하이라이트, 이미지 블렌딩 등 고급 그래픽 처리 완료.

3.  **스타일링:**
    - 전문적인 느낌의 다크 UI.
    - 가독성을 극대화한 아티클 섹션 스타일링 (좌측 보더, 박스형 리스트 등).
