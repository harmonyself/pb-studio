# Blueprint: 유썸생 (유튜브 썸네일 종합 플랫폼)

## 1. 프로젝트 개요

'유썸생'은 단순한 제작 도구를 넘어 유튜브 크리에이터와 디자이너를 위한 **종합 썸네일 생태계**입니다. 고성능 썸네일 생성기, 전문가의 제작 노하우, 실시간 트렌드 분석, 명예의 전당(어워즈), 그리고 디자인 의뢰 서비스까지 한곳에서 제공하는 올인원 플랫폼입니다.

## 2. 핵심 서비스 및 기능

- **썸네일 생성기 (Tool):** 
    - AI 자동 배경 제거(누끼), 3분할 배경 합성, 텍스트 하이라이트.
    - 유튜브 영상 링크를 통한 썸네일 추출 기능.
- **제작 노하우 (Guide):** 
    - 전략, 디자인 심리학, 툴(포토샵, 캔바, PPT) 사용법 등 16개 이상의 심층 아티클.
    - 데이터 기반의 동적 블로그 시스템.
- **트렌드 분석 (Trends):** 
    - 글로벌 및 국내 인기 영상의 썸네일 스타일 실시간(랜덤 시뮬레이션) 분석.
    - 직접 분석 도구를 통한 벤치마킹 지원.
- **명예의 전당 (Awards):** 
    - 연간 누적 조회수 기준 TOP 10 썸네일 랭킹 시스템.
- **작업 의뢰 (Jobs):** 
    - Formspree 연동을 통한 실제 작동하는 디자인 의뢰 시스템.

## 3. 기술 스택 및 구조

- **Frontend:** HTML5 (Semantic), CSS3 (Modern/Responsive), JavaScript (ES6+).
- **Architecture:** 
    - Multi-page Application (MPA) 구조.
    - `article_data.js`, `youtube_data.js`를 활용한 데이터 중심 렌더링.
- **Libraries:** `@imgly/background-removal` (AI), Canvas API (Graphics).
- **Integration:** Formspree (Backend-as-a-service for forms).

## 4. 디자인 및 스타일 가이드

- **레이아웃:** 상단 통합 내비게이션(GNB)을 통한 유기적 연결.
- **디자인 테마:** 세련된 다크 모드 기반의 고대비 UI, 모바일 최적화 레이아웃.
- **타이포그래피:** `Pretendard` 폰트를 활용한 전문적인 블로그 스타일 및 카드 UI.

## 5. 구현 현황 (플랫폼 고도화 완료)

1.  **플랫폼 전환:** 단순 도구에서 5개 핵심 메뉴를 가진 포털 사이트로 확장 완료.
2.  **데이터 동적화:** 트렌드 및 어워즈 데이터의 무작위 셔플 및 정렬 로직 구현.
3.  **콘텐츠 확충:** 16개의 상세 가이드 및 실제 유튜브 성공 사례 DB 구축.
4.  **실무 연동:** 실제 작동하는 의뢰 폼 및 유튜브 추출 기능 탑재.
