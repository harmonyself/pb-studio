# Blueprint: 유튜브 썸네일 생성기

## 1. 프로젝트 개요

'꿈의 그라운드 경남' 스타일의 유튜브 강연 썸네일을 쉽고 빠르게 제작할 수 있는 웹 애플리케이션입니다. 사용자는 강연자 사진, 배경, 로고, 텍스트 등 각 요소를 직접 업로드하고 편집하여 완성도 높은 썸네일을 생성하고 다운로드할 수 있습니다.

## 2. 핵심 기능

- **템플릿 기반 제작:** 제공된 '꿈의 그라운드 경남' 썸네일 템플릿을 기반으로 합니다.
- **실시간 캔버스 편집:** 모든 변경사항은 캔버스에 실시간으로 반영되어 직관적인 편집이 가능합니다.
- **요소별 커스터마이징:**
    - **강연자 이미지:** 좌측에 배치되며, 위치(X, Y) 및 크기(Scale) 조절이 가능합니다.
    - **배경 이미지:** 전체 배경으로 사용되며, 직접 업로드하거나 AI 생성 기능을 사용할 수 있습니다.
    - **로고 이미지:** 우측 상단에 배치되며, 위치 및 크기 조절이 가능합니다.
    - **강연자 정보:** 이름과 소속 텍스트를 입력하고 위치와 크기를 조절할 수 있습니다.
    - **강조 문구:** 2줄의 굵은 강조 문구를 입력하고, 위치, 크기, 특정 단어 하이라이트 색상 등을 편집할 수 있습니다.
- **AI 배경 생성 연동:** 배경 이미지가 없을 경우, 버튼 클릭으로 AI 이미지 생성을 요청하는 기능을 제공합니다 (현재는 UI만 구현).
- **고해상도 다운로드:** 최종 결과물은 1280x720 해상도의 PNG 파일로 다운로드할 수 있습니다.

## 3. 기술 스택

- **HTML5:** `<canvas>`, `<input type="range">` 등 시맨틱한 마크업
- **CSS3:** Flexbox, Grid, CSS 변수 등을 활용한 모던하고 반응형적인 스타일링
- **JavaScript (ES6+):** Canvas API를 활용한 동적 이미지 및 텍스트 렌더링, 이벤트 처리

## 4. 디자인 및 스타일 가이드

- **레이아웃:** 직관적인 2단 레이아웃 (좌: 컨트롤 패널, 우: 캔버스 미리보기)
- **색상 팔레트:**
    - Primary: `#FF0000` (유튜브 레드)
    - Background: `#181818` (다크 모드)
    - Text: `#FFFFFF`
    - Highlight: `#FFFF00` (강조 문구 하이라이트)
- **폰트:** 시인성이 높은 `Pretendard` 또는 `Noto Sans KR` 같은 산세리프 폰트 사용
- **인터랙션:** 슬라이더와 컬러 피커를 통해 부드럽고 직관적인 사용자 경험 제공

## 5. 현재 구현 계획

1.  **HTML (`index.html`) 구조 변경:**
    - 강연자, 로고, 이름, 소속, 강조 문구 등 모든 요소를 위한 `input` 필드 추가.
    - 각 요소의 위치(X, Y), 크기(Scale)를 조절할 수 있는 `<input type="range">` 슬라이더 추가.
    - 강조 문구의 하이라이트 색상을 선택할 수 있는 `<input type="color">` 추가.
    - AI 배경 생성 버튼 추가.

2.  **CSS (`style.css`) 개선:**
    - 유튜브 스타일과 유사한 세련된 다크 모드 디자인 적용.
    - 컨트롤 패널 UI를 사용하기 쉽게 개선.
    - 입력 필드, 버튼, 슬라이더 등 모든 UI 요소에 커스텀 스타일 적용.

3.  **JavaScript (`main.js`) 로직 재구성:**
    - 모든 입력 요소에 대한 참조 변수 선언.
    - `drawCanvas()` 함수를 중심으로 로직 통합.
    - 이미지(배경, 강연자, 로고)와 텍스트(이름, 소속, 강조 문구)를 캔버스에 그리는 함수 구현.
    - 슬라이더 및 입력 값 변경 시 `drawCanvas()` 함수를 즉시 호출하여 실시간 미리보기 구현.
    - 강조 문구에서 특정 단어(예: `(강조)단어`)를 인식하여 하이라이트 처리하는 로직 추가.
    - 다운로드 기능 구현.
